# gravity-simulator

[![Live Demo](https://img.shields.io/badge/🚀_Live_Demo-Visit-blue?style=for-the-badge)](https://mayank-dev-15.github.io/gravity-simulator)
[![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)](https://opensource.org/licenses/MIT)
![Language](https://img.shields.io/badge/Language-HTML/CSS/JS-green)

N-body orbital mechanics simulator with solar system, binary stars, three-body problem, and realistic Newtonian gravity.

`physics` `gravity` `simulation` `canvas` `space`

---

## ✨ Features

- Newtonian gravity (F = G·m₁·m₂/r²)
- Click & drag to add bodies with initial velocity
- 5 presets (Solar System, Binary Star, Three-Body, Galaxy Collision, Figure-8)
- Adjustable gravitational constant
- Time speed control (pause, slow-mo, fast-forward)
- Trail/orbit path visualization
- Collision detection with momentum-conserving merging
- Mass, velocity, and energy display for selected body
- Parallax star field background
- Export/Import simulation state as JSON

---

## 💡 Usage

- Click and drag to create a body (drag sets velocity)
- Space to pause/resume
- Scroll to zoom, right-drag to pan
- T to toggle trails
- 1-5 to load presets
- Delete to remove selected body
