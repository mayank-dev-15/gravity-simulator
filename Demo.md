## About

**gravity-simulator** is a n-body orbital mechanics simulator.

### Key Features

- Newtonian gravity (F = G*m1*m2/r2)
- Click & drag to add bodies with initial velocity
- 5 presets (Solar System, Binary Star, Three-Body, Galaxy Collision, Figure-8)
- Adjustable gravitational constant
- Time speed control
- Trail/orbit path visualization
- Collision detection and merging
- Mass and velocity display
- Parallax star field background

## Controls & Usage

| Control | Action |
|---------|--------|
| `Click+drag` | Set body mass and velocity |
| `Space` | Pause/resume |
| `Scroll` | Zoom |
| `Right-drag` | Pan |
| `T` | Toggle trails |
| `1-5` | Load presets |
| `Del` | Remove selected |
